Firebase Dynamic Links

Firebase Dynamic Links are links that work the way you want, on multiple platforms, and whether or not your app is already installed. As a result, I used this feature in one of my company's projects to share an app via a web portal via QR code.

Selenium IDE

Selenium IDE is a Chrome, Firefox and Edge plugin which records and plays back user interactions with the browser. Use this to either create simple scripts or assist in exploratory testing. Using Javascript and Python scripts, I constructed a sample for automation testing.

Amazon Web Services

Following an internship, I began working on a live project that is now hosted on Amazon Cloud. As a result, I used S3 Bucket for storage, Elastic Beanstalk to operate and manage the app, and CodePipeline to release software using Continuous Delivery.

Elastic Search

A set of premium tools for building machine learning-powered search into your websites, custom applications, and workplaces. So I used this for searching functionality which client wants to implement in one of my company's projects. For that we connected elastic search with AWS RDS real data

GraphQL

GraphQL provides a complete description of the data in your API. I explore this through practice demo

MongoDB

MongoDB is a source-available cross-platform document-oriented database program for high-volume storage. Implemented this as a database in one of my demo project.

Stripe

Stripe provides all the tools you need to accept payments online and in-person from customers around the world. So, I used Stripe as a payment gateway.

Docker

Docker allows you to run containers locally, eliminating disparity between your development and production environments, and everything in between. Implemented this in one of my demo project.

Heroku

Heroku is used to host and run our web application and background processes. As a result, I hosted one of my web application on Heroku.

Husky

Husky is a tool that allows us to easily wrangle Git hooks and run the scripts we want at those stages. So I utilised Husky to add Pre Commit Hooks for Prettier code formatting and Automatic execute jest test cases.

Handlebars

Handlebars is a simple templating language. It uses a template and an input object to generate HTML or other text formats. As a result, I utilised Handlebars for email templates and as a frontend language for the frontend.
